# Data Download Instructions

## May CPS (1973–1978)
Download from: https://www.nber.org/research/data/current-population-survey-cps-may-extracts-1969-1987
- Download cpsmay1973.dta through cpsmay1978.dta
- Place all files in input/

## ORG CPS (1979–1991)
Download from: https://data.nber.org/morg/annual/
- Download morg1979.dta through morg1991.dta
- Place all files in input/
